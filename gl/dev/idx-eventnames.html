Names of Events

2026-02-18

Tagging of events named in the transcribed text or elsewhere, usually for the purposes of indexing or searching the corpus.

  • TEI definition: eventName ; EpiDoc-specific customization: eventName
  • TEI definition: event ; EpiDoc-specific customization: event
  • TEI definition: listEvent ; EpiDoc-specific customization: listEvent

Event names may be tagged using the eventName element. It may be advisable to attach a nymRef or key attribute point to an authority list in a listEvent elsewhere in the project, or use the ref attribute point to an external authority.

<eventName nymRef="eventslist.xml#bellumDacicum">bellum Dacicum</eventName>
<!-- CIL VIII 1026, EDH HD051091 -->
<eventName ref="https://www.wikidata.org/wiki/Q650197">Lupercalia</eventName>
<!-- CIL IV 06882 -->
