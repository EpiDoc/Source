Inventory numbers and repository

Recording the holding institution of a manuscript or text-bearing object and inventory number in a museum or library catalogue, if applicable; the common name or label of the object if it has no official inventory number.

  • TEI definition: msIdentifier
  • TEI definition: msName
  • TEI definition: repository
  • TEI definition: idno
  • TEI definition: altIdentifier

The TEI requires some kind of manuscript identifier (<msIdentifier>) to be encoded as part of the manuscript description in any edition, either institution plus identifier (<repository>+<idno>), or a generic manuscript name (<msname>). For epigraphy, where many objects are not held or catalogued by institutions but may be in situ, stray, in private collections or even lost, we may need to apply these definitions loosely in order to complete these elements.

<msIdentifier>
 <repository>National Preserve of Tauric Chersonesos (НЗХТ)</repository>
 <idno>80/36504</idno>
</msIdentifier>
<msIdentifier>
 <msName>The Column of Trajan</msName>
</msIdentifier>
