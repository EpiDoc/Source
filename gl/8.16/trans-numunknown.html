Numbers/numerals of Unknown Value

  • TEI definition: num

The numeral should be placed inside a <num>, but where its value is unknown, no value can be given.

‘’Silvio Panciera ...Josh Sosin
<num/>
<num>
 <gap
   reason="lost"
   atLeast="1"
   atMost="4"
   unit="character"/>
</num>
