Fractions of Unknown Value

  • TEI definition: num

The fraction should be placed inside a <num>, but where its value is unknown, no value can be given. It should be given a type of "fraction".

‘’Silvio Panciera ...Josh Sosin
<num type="fraction">
 <gap
   reason="lost"
   atLeast="2"
   atMost="4"
   unit="character"/>
</num>
