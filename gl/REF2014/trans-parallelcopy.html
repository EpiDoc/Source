Parallel Text

Text restored from parallel copy

  • TEI definition: supplied ; EpiDoc-specific customization: supplied

Where text is restored by the editor from a parallel copy, this should be indicated by using the usual supplied tag, but with an evidence with a value of "parallel".

‘[abc]’Silvio Panciera VIII.1Josh Sosin
<supplied reason="undefined" evidence="parallel">abc</supplied>
