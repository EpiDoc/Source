Describing past and present locations of the inscribed object

Implementation of the guidance in this section is required for aiegl conformance.

Rigorous epigraphic publications provide detailed information on the geographic aspects of each inscription, insofar as these are known to the editor or can be recovered. Apart from any geographic content inherent or implicit in the text itself (e.g., place names, for which see Geographic Names), epigraphic editions normally provide users with information about the modern place of finding (and the name of any associated ancient settlement), as well as the last known or observed location (complete with any appropriate inventory numbers). EpiDoc documents must include this information, insofar as it is known, if they aim for EAGLE-conformance.

Editors may include this information in their documents, either as <keyword> <term>s in the <textClass> portion of the <teiHeader> element, or, in the form of prose or text paragraphs, encapsulated in a <div type="description"> element. If an editor wishes to use multiple <div type="description"> elements for different descriptive aspects of the edition, the description element containing the location information should be specifically indicated by using the “subtype” attribute, and giving it a value of “location”.

Whether location information is placed in the <teiHeader> or in a <div type="description">, the following tagging guidance must be followed closely for EAGLE conformance:

EpiDoc Encoding

  1. Modern settlement where the inscription was found: <placeName type="modernFindspot" key="http://www.geonames.org/88319/banghazi.html">Benghazi</placeName>
  2. Ancient name associated with the modern place of finding: <placeName type="ancientFindspot" key="http://pleiades.stoa.org/places/373757">Berenike</placeName>
  3. Greater detail concerning the modern place of finding: <rs type="found">Via Mentana, Columbarium B</rs>
  4. Modern location where the item was last observed: <rs type="lastLocation">Speed Museum, Louisville, Kentucky, USA; <rs type="InvNo">BQ773127</rs></rs>

EpiDoc appearance with standard XSLT

  1. Modern settlement where the inscription was found: Benghazi
  2. Ancient name associated with the modern place of finding: Berenike
  3. Greater detail concerning the modern place of finding: Via Mentana, Columbarium B
  4. Modern location where the item was last observed: Speed Museum, Louisville, Kentucky, USA; BQ773127

Additional guidance

Although it is not required for EAGLE conformance, editors who wish to encode information about the original location of the inscription may use, e.g., <rs type="originalLocation">theater</rs>.

Wherever possible, geographic information should be controlled using the “key” attribute on the appropriate <rs> or <placeName> elements as outlined in the Guidelines section on Geography.
