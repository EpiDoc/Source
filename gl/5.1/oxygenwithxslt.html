Instructions of how to use the epidoc XSLT stylesheets with your own epidoc files. This was written using Oxygen v7.1

  • With the XML file open in Oxygen, click the "Apply transformation scenario" button.
  • In the "Configure transformation scenario" window, click on "new" to create a new scenario.
  • In the XSLT tab, enter the path for force-namespace-tei.xsl into XSL URL
  • Click on "Parameters", "Add". Enter in "Name", 'standalonecss'. In "Value", enter the path for the location of epidocscreen.css.
  • Back in the XSLT tab, click on "Additional XSLT stylesheets", "Add". Enter the path for epidoc.xsl
  • Return to the XSLT tab, in the Output tab, select "Prompt for file" and "Open in browser".
  • Returning to the original "Configure transformation scenario" window, click "Transform now".
  • Save to a desired location and see the results!
