Crosswalker: A Tool for Converting EpiDoc to Other Formats

Crosswalker

“Crosswalker” is a software toolset for converting EpiDoc XML documents to other formats; i.e., to comma-separated-value (CSV) text, or to XML governed by another schema. Users create an XML file (the “crosswalk file”) that maps (or describes) the relationships between components of the EpiDoc XML file and components of the targeted format. An XSL transform (generator/crosswalker.xsl) interacts with the crosswalk file to produce a separate, custom XSL transformation file (the “transform”) that can then be used to convert EpiDoc documents to the desired format.

Getting Started with Crosswalker

  1. Write (or adapt) a crosswalk XML file
  2. Execute the crosswalker.xsl file against the crosswalk XML file
  3. Use the resulting, custom XSL file to convert your EpiDoc documents
