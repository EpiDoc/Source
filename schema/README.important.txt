The /latest/ directory inside /schema/ is the "latest
stable (numbered) release" of the EpiDoc Schema
(copy of /8.18/ or whatever the latest is).
The contents of this directory should ONLY be updated
or overwritten when a formal EpiDoc release is being made.

Any other updates to the EpiDoc Schema, whether
experimental or final, intended for the next release,
should be uploaded to /dev/ only until the
full 14-day release process has occurred.
