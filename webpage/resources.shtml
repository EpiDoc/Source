EpiDoc Resources and Tools

Members of the EpiDoc Community have created, and maintain, a range of guidelines, software tools and other resources to facilitate the creation, dissemination and exploitation of EpiDoc works. This page provides basic information about each of the major resource packages, with links to complete documentation, download locations and online copies.

The EpiDoc Guidelines

The EpiDoc Guidelines is a prose document intended to provide complete guidance for the application of TEI markup to the preparation of digital epigraphic editions and similar documents. They also provide documentation for all of the resources described on this page.

The Guidelines are collaboratively written by the developers in the EpiDoc Community. At present, they are authored under the auspices of a slightly hacked version of the same TEI customization that is distributed as the EpiDoc DTD; however, work is underway to replace this method with a custom RelaxNG schema that will also function as a TEI P5 ODD from which a future P5-compliant version of the EpiDoc schema will be generated (see the schema-gl module in the EpiDoc CVS Repository on SourceForge.

Copies of The EpiDoc Guidelines can be obtained in one of the following ways:

  • Current, stable “release” versions:
    • Online (in HTML), courtesy of the Stoa Consortium (coming soon)
    • As a downloadable package of XML files, courtesy of SourceForge
    • As a downloadable package of HTML files, courtesy of SourceForge (coming soon)
  • Raw, development versions:
    • Online (in HTML), courtesy of the Stoa Consortium
    • Via checkout from the EpiDoc CVS Repository, courtesy of SourceForge (module name: guidelines)

The EpiDoc Document Type Definition (DTD)

At present, the EpiDoc Guidelines make recommendations in accordance with the P4 version of the TEI Guidelines. A customized DTD, created using the TEI Pizza Chef, provides for coordinating editorial control and document validation.

Copies of the EpiDoc DTD can be obtained in one of the following ways:

  • Current, stable “release” versions:
    • Online, courtesy of the Stoa Consortium (coming soon)
    • As a downloadable package, courtesy of SourceForge
  • Raw, development versions:
    • Online, courtesy of the Stoa Consortium (coming soon)
    • Via checkout from the EpiDoc CVS Repository, courtesy of SourceForge (module name: dtd)
